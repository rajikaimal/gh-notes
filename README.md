# inq

Take notes on GitHub with inq CLI

## Install

```

```

## Usage

### Configure

- Create a repository
- Provide repository name to inq

```
$ inq config [githubRepositoryUrl]
```

### Save note by date

```
$ inq save
```

### Save note by topic

```
$ inq save --topic=[topicName]
```

### Save note by name

```
$ inq save --name=[fileName]
```

### Push to GitHub

```
$ inq push
```
