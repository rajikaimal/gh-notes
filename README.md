# inq

Take notes on GitHub with inq CLI

## Install

```

```

## Usage

### Configure

- Create a repository
- Provide repository name to inq

```
$ inq --github=[githubRepositoryUrl] config
```

### Save note by date

```
$ inq save
```

### Save note by topic

```
$ inq --topic=[topicName] save
```

### Save note by name

```
$ inq --name=[fileName] save
```

### Push to GitHub

```
$ inq push
```
